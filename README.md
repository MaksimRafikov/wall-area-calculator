# Калькулятор площади стен

Веб-приложение для сметы **внутренней отделки**: площадь стен по комнатам (с учётом общих перегородок), вычет проёмов, отдельно **откосы окон** и **откосы дверей/прочих проёмов**.

## Онлайн-версия

После настройки GitHub Pages:

**https://maksimrafikov.github.io/wall-area-calculator/**

(Первый деплой может занять 2–5 минут после push в `main`.)

## Запуск на своём компьютере

```powershell
cd "путь\к\папке\проекта"
npm install
npm run dev
```

Откройте в браузере адрес из терминала (обычно http://localhost:5173).

Сборка:

```powershell
npm run build
npm run preview
```

## Возможности

- Ввод размеров в **миллиметрах**, итоги в **м²**
- Помещения → стены → проёмы (окно, дверь, арка, прочее)
- Пресет **«Видинеевский»** (кнопка в интерфейсе)
- Таблица для сметы и **экспорт CSV**
- Данные сохраняются в браузере (localStorage)
